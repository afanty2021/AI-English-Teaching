## 1. 系统概述

### 1.1 功能简介

学生端是 AI 赋能英语教学系统的学习平台，为英语学习者提供以下功能：

| 功能模块 | 核心能力 |
|---------|----------|
| **学习仪表板** | 学习概览、每日任务、进度追踪 |
| **课程学习** | 系统化课程内容、知识点学习 |
| **练习中心** | 多题型练习、AI批改反馈 |
| **错题本** | 错题收集、AI分析、复习提醒 |
| **口语练习** | AI口语陪练、发音评测 |
| **AI对话** | 沉浸式英语对话、零压力练习 |
| **学习进度** | 个人学习数据统计、能力成长追踪 |
| **学习报告** | 周报/月报、学习分析、建议生成 |

### 1.2 访问方式

**Web 端**：
1. 打开浏览器访问系统地址
2. 使用学生账号登录
3. 进入学习平台

## 2. 快速入门

### 2.1 首次登录

**步骤 1：登录系统**
- 访问系统登录页面
- 输入学生账号和密码
- 点击「登录」按钮

**步骤 2：完善个人资料（首次）**
- 设置头像（可选）
- 确认年级信息
- 设置目标考试类型
- 设定目标分数

**步骤 3：初始诊断**
- 首次登录会引导完成初始诊断
- AI 评估你的英语水平
- 生成个性化知识图谱
- 基于诊断结果推荐学习路径

### 3.3 今日任务

系统根据你的学习计划和薄弱点，每日推荐学习任务。

#### 任务类型

| 类型 | 说明 | 预计时长 |
|------|------|----------|
| 课程学习 | 推荐的学习内容 | 10-20分钟 |
| 练习巩固 | 薄弱知识点练习 | 10-15分钟 |
| 口语对话 | AI口语陪练 | 5-10分钟 |
| 错题复习 | 待复习的错题 | 5-10分钟 |

#### 完成任务

1. 点击任务项进入对应页面
2. 完成学习/练习活动
3. 系统自动标记任务完成
4. 更新学习统计

### 3.4 继续学习

显示最近学习的课程和进度：

- 点击「立即继续」继续上次学习
- 点击「查看全部」浏览所有课程
- 支持跨设备同步学习进度

### 4.6 学习状态标记

每个单词学习后可标记状态：

| 标记 | 说明 | 复习安排 |
|------|------|----------|
| **掌握** | 完全熟悉此单词 | 较晚复习 |
| **模糊** | 有印象但不牢 | 近期复习 |
| **不认识** | 首次学习/忘记 | 优先复习 |

系统会根据标记自动安排复习计划。

4. 点击链接重置密码

### Q2: 初始诊断需要多长时间？

**答**：初始诊断通常需要 5-10 分钟，包括：
- 英语水平评估问卷（2-3分钟）
- AI综合分析（1-2分钟）
- 生成知识图谱（1-2分钟）

### Q3: 知识图谱多久更新一次？

**答**：
- **自动更新**：每次完成练习后自动微调
- **定期复盘**：建议每周查看更新
- **重新诊断**：如感觉水平变化，可手动重新诊断

### Q4: 口语练习可以离线使用吗？

**答**：目前暂不支持离线，需保持网络连接。

### Q5: 错题本会自动提醒复习吗？

**答**：是的，系统会根据艾宾浩斯遗忘曲线自动安排复习计划，并在首页「今日任务」中提醒。

### Q6: 对话记录会保存多久？

**答**：
- 对话记录永久保存
- 可手动删除不需要的对话
- 导出后可本地保存

### Q7: 如何查看我的CEFR等级？

**答**：
- 首页仪表板显示当前等级
- 知识图谱页面有详细说明
- 诊断报告中有等级评估依据

### Q8: 学习数据会同步吗？

**答**：是的，所有学习数据实时同步，跨设备登录即可查看。

### Q9: 如何设置学习目标？

**答**：
1. 进入「设置」→「学习目标」
2. 选择目标考试类型
3. 设定目标分数
4. 设置每周学习时长目标
