## 1. 系统概述

### 1.1 功能简介

教师端是 AI 赋能英语教学系统的核心管理平台，为英语教师提供以下功能：

| 功能模块 | 核心能力 |
|---------|----------|
| **学生管理** | 学生列表查看、知识图谱诊断、批量诊断操作 |
| **班级管理** | 班级创建、学生分配、学习数据汇总 |
| **题库管理** | 题库创建、题目编辑、批量导入导出 |
| **教案管理** | AI教案生成、教案编辑、模板库管理 |
| **教案导出** | PDF/Word/PPTX格式导出、批量导出 |
| **学习报告** | 班级报告生成、学生个体报告、学习分析 |
| **AI备课** | 智能备课建议、教学设计生成 |

### 1.2 访问入口

1. 打开浏览器访问系统地址
2. 使用教师账号登录
3. 进入教师端工作台

## 2. 快速入门

### 2.1 首次登录

**步骤 1：登录系统**
- 访问系统登录页面
- 输入教师账号和密码
- 点击「登录」按钮

**步骤 2：完善个人资料（首次）**
- 头像和昵称设置
- 教授科目选择
- 教学简介填写

**步骤 3：了解功能布局**
- 熟悉侧边栏导航
- 了解快捷操作入口
- 设置通知偏好

### 2.2 快捷操作

| 操作 | 路径 | 说明 |
|------|------|------|
| 查看班级概况 | 仪表板 → 班级概况卡片 | 快速了解班级学习数据 |
| 诊断学生 | 学生管理 → 选择学生 → 诊断 | AI分析学生英语水平 |
| 生成教案 | AI备课 → 输入需求 → 生成 | AI辅助教案设计 |
| 导出报告 | 学习报告 → 选择报告 → 导出 | 多种格式导出 |

#### CEFR 等级说明

| 等级 | 名称 | 对应能力描述 |
|------|------|-------------|
| **A1** | 入门级 | 能理解和使用日常熟悉的表达方式 |
| **A2** | 初级 | 能理解句子和相关领域的短语 |
| **B1** | 中级 | 能处理旅行、学校等场景的大多数情况 |
| **B2** | 中高级 | 能理解具体或抽象主题的复杂文本 |
| **C1** | 高级 | 能流利自如地表达各种主题 |
| **C2** | 精通级 | 能轻松理解几乎所有内容 |

### 4.4 学生诊断

#### 单个学生诊断

**适用场景**：新学生入学、需要重新评估学生水平

**操作步骤**：

1. 在学生列表中找到目标学生
2. 点击「诊断」按钮
3. 在弹出的确认对话框中确认
4. 等待诊断完成（通常 30-60 秒）
5. 诊断完成后自动展示知识图谱

#### 批量学生诊断

**适用场景**：学期初统一诊断、班级整体评估

**操作步骤**：

1. 在学生列表中勾选需要诊断的学生
2. 点击「批量诊断」按钮
3. 查看诊断预览（选中数量）
4. 点击「开始诊断」
5. 查看诊断进度和结果统计

2. 点击「添加学生」
3. 选择要添加的学生（可多选）

## 12. 常见问题

### Q1: 学生无法登录？

**检查项**：
- 账号密码是否正确
- 账号是否已激活
- 网络连接是否正常

**解决方案**：联系管理员检查学生账号状态。

### Q2: 诊断结果显示异常？

**可能原因**：
- 学生练习数据不足
- AI服务暂时不可用
- 网络超时

**解决方案**：
- 确保学生完成足够练习
- 稍后重试诊断
- 检查系统通知

### Q3: 教案导出失败？

**可能原因**：
- 文件格式不支持
- 教案内容过于复杂
- 网络超时

**解决方案**：
- 尝试其他导出格式
- 分拆复杂教案
- 重新导出

### Q4: 批量诊断进度卡住？

**处理方式**：
- 点击「取消」停止当前任务
- 重新发起诊断（系统会自动跳过已完成的）
- 检查后端服务状态

### Q5: 学习报告数据不准确？

**检查项**：
- 学生是否正常使用系统
- 数据同步是否完成
- 是否跨周期切换

**解决方案**：
- 等待数据同步（通常 1-2 小时）
- 重新生成报告
- 联系技术支持

### Q6: 如何备份教案？

**方法**：
1. 导出所有教案（批量导出）
2. 下载到本地保存
3. 建议定期（如每月）备份一次

### Q7: 忘记密码？

**重置流程**：
1. 点击登录页「忘记密码」
2. 输入注册邮箱
3. 接收重置邮件
4. 点击链接重置密码
