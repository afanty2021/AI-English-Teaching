## 快速导航

### 教师用户

```
1. 阅读 [教师端使用指南](teacher-guide.md)
2. 了解系统功能 → 仪表板、学生管理、教案管理
3. 进阶功能 → 教案导出、学习报告、AI备课
```

### 学生用户

```
1. 阅读 [学生端使用指南](student-guide.md)
2. 开始学习 → 课程学习、练习中心
3. 提升能力 → 口语练习、AI对话
4. 查看进度 → 学习报告、进度追踪
```

### 运维人员

```
1. 阅读 [部署运维手册](deployment-guide.md)
2. 环境准备 → Docker部署、配置管理
3. 日常运维 → 服务管理、监控、备份
4. 故障排查 → 常见问题、性能调优
```
